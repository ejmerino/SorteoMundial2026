# **App Name**: Copa Draw Simulator

## Core Features:

- Simulate Draw: Simulate the World Cup 2026 draw based on defined rules (pot distribution, geographic restrictions, host country assignments).
- Rule Validation: Validate the draw results to ensure all restrictions and rules are adhered to (e.g., maximum teams per confederation in a group).
- Visual Display: Display the draw results in a clear and intuitive format, showing team flags, group assignments, and pot numbers.
- Host Nation Placement: Automatically place Mexico in Group A, Canada in Group B, and USA in Group D as hosts.
- Data Population: Use pre-loaded (burned-in) data for participating teams, their FIFA rankings, and confederation assignments.

## Style Guidelines:

- Primary color: Dark blue (#1A237E) to evoke a sense of prestige and seriousness, reflecting the importance of the World Cup draw.
- Background color: Very light blue (#E3F2FD) for a clean, modern, and unintrusive backdrop that emphasizes legibility.
- Accent color: Orange (#E65100), a complementary color, used sparingly for CTAs to draw attention in a way that is friendly, confident, and sporty.
- Body and headline font: 'Inter', a sans-serif font with a modern and neutral aesthetic suitable for on-screen use.
- Use flags of countries to visually represent teams, enhancing user immersion.
- Responsive layout using Tailwind CSS for optimal viewing across different devices (desktops, tablets, and smartphones).
- Subtle transitions and animations to enhance the user experience during the draw simulation process.
